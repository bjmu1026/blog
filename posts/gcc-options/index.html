gcc常用命令选项

  • -ansi 只支持ANSI标准的C语法。
  • -O 优化
    • -O0不进行优化
    • -O1 优化
    • -O2 进一步优化
    • -O3 最大优化
  • -shared –fPIC 生成共享目标文件，通常用在建议共享库
  • -static 禁止使用共享连接
  • -D DEBUG 定义DEBUG宏
  • -U DEBUG 取消DEBUG宏
  • -w 不生成任何警告信息
  • -Wall 生成所有警告信息
  • -E 只激活预处理
  • -S 只激活预处理和编译，把文件编译成汇编代码
  • -c 只激活预处理、编译和汇编，把文件做成obj文件
  • -include file 包含某个代码
  • -Idir 在所制定的目录中查找include 的文件
  • -I- 取消前一个参数的功能
  • -idirafter dir 在-I目录里面查找失败，再进入到这个目录查找
  • -llibrary 制定编译时使用的库
  • -Ldir 制定编译时，搜索库的路径
  • -g 在编译时产生调试信息

库生成

  • 静态库

    gcc -c file1.c -o file1.o
gcc -c file2.c -o file2.o
ar rcs libmylib.a file1.o file2.o

gcc mian.c -L. -lmylib -o program

  • 动态库

    gcc -fPIC file1.c -o file1.o
gcc -fPIC file2.c -o file2.o
gcc -shared -o libmylib.so file1.o file2.o
gcc main.c -L. -lmylib -o program
