ip命令

  • ip命令是新一代网络配置工具，旨在替代ifconfig、route、arp等命令。

核心模块ip addr

  • 查看所有网络接口的ip信息

    ip addr show  #显示所有网络接口的ip配置ipv4 ipv6 mac 接口状态
ip addr show dev enp10s0  #仅显示eth0的ip信息

  • 删除网络接口的Ip地址

    ip addr del 192.168.0.2/24 dev eth0 #删除eth0接口的ip地址
ip addr flush dev eth0 #删除该接口的所有ip地址

核心模块ip link

  • 查看所有网络接口的链路状态

    ip link #查看所有接口信息mac state mtu
ip link show dev eth0 #查看指定接口信息

  • 启用和禁用网络接口

ip link set dev enp10s0 up  #启用接口
ip link set dev enp10s0 down #禁用接口
ip l show dev enp10s0 | grep state

  • 修改接口mtu大小

    ip link set dev enp10s0 mtu 1500 #设置接口MTU MAXimum transmission unit

  • 修改mac地址

    ip link set dev eth0 down #禁用接口
ip link set dev eth0 address 00:11:22:33:44:55 #更改mac地址
ip link set dev eth0 up   #启用接口
ip link show dev eth0 | grep link/ether

核心模块ip route

  • 查看系统当前路由表

    ip route show

  • 添加静态路由

    ip route add 192.168.1.0/24 via 192.168.0.254 dev eth0 #手动添加网关

  • 删除静态路由

    ip route del 192.168.1.0/24 #删除该网段的路由
ip route flush all # 删除所有路由

核心模块ip netns(网络命名空间)

  • 查看当前系统的网络命名空间

    ip netns list #查看所有已创建的网络命名空间

  • 创建/删除网络命名空间

    ip netns add ns1 
ip netns delete ns1

  • 在命名空间内执行网络命令(核心用法)

    ip netns exec ns1 ip addr show #默认接口只有lo回环口
ip netns exec ns1 ip link set dev lo up #启用ns1内的回环口,默认禁用

ip link add veth-ns1 type veth peer name veth-main #创建一对veth设备veth-ns1 和 veth-main
ip link set veth-ns1 netns ns1 
ip netns exec ns1 ip addr add 10.0.0.1/24 dev veth-ns1
ip link set veth-main up 
ping -c 2 10.0.0.1 #测试主机与ns1的连通性
